Creating a new design

Create a new design in RSF by selecting File | New design or by clicking the button on the main tool bar.

The Add Artwork dialog box will be displayed. If you want to add an artwork to the design at this stage, enter the sight line measurements of the design. Alternatively, if you want to select a picture to use for the artwork, select it from the drawing combo box or specify a color to use for the artwork instead. When done, click on OK.

If you do not want to add an artwork to the design, click on Cancel. An artwork can be added at a later stage by selecting Designs | Add | Add Artwork, or by clicking on the add artwork button.
