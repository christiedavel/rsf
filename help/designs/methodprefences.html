Changing method preferences of a specific design

When a design is created, it inherits the latest set method preferences. If the method preferences for the specific design needs to be changed without affecting the set preferences of all other designs, select the bottom switch () on the design details panel. The local method preferences panel will be displayed where you can make the changes as required. The preferences in Preferences | Method Preferences will remain unchanged.

If at a later stage you want to update the method preferences to the latest settings, simply follow the same procedure as described in Updating preferences of a design.
