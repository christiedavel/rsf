Inserting items into a design

1. Boards

To insert a board in a design, select an existing design item by clicking on the item's area in the display or by selecting it in the measurements table. Click on the insert board button or select Designs | Insert | Insert board. The new board will be the inserted to the inside of the selected item.

The insert board dialog will be displayed. Select the itemcode or owncode of the preferred board. Specify the face values of the new board and click on OK.

If all the face values are the same, select the match all check box (selected by default) and only enter the top face. All the other faces will be the same as the top face.

If the board is floating,select the floating board check box. Enter the offset values as required. Again, if all the offsets are the same, select the match all check box and only specify the top offset.

If you want to specify custom overlaps for the board, click on the custom overlaps check box and enter the overlaps.

If no item is selected, the new board will be the outermost item in the design.


2. Frames

To insert a frame in a design, select an existing design item by clicking on the item's area in the display or by selecting it in the measurements table. Click on the insert frame button or select Designs | Insert | Insert frame. The new board will be the inserted to the inside of the selected item.

The Insert frame dialog will be displayed. Select the itemcode or owncode of the preferred frame.

If the frame is floating, select the floating frame check box. Enter the offset values as required. If all the offsets are the same, select the match all check box and only specify the top offset.

To insert a frame into a design, there must be at least an artwork or board already in the design.

The Insert slip dialog will be displayed. Select the itemcode or owncode of the preferred slip.

To insert a slip into a design, there must be at least an artwork or board already in the design.

The Insert foil dialog will be displayed. Select the itemcode or owncode of the preferred foil.
