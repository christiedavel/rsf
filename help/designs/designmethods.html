Design Methods

1. Calculated overlap

Use the calculated overlap when your stock is low. The calculated overlap is the smallest overlap that can be used without causing technical problems when framing and is the most cost effective.


2. Fixed border overlap

Use fixed border overlap to expedite board cutting. In this case all the bottom boards are cut at the same width. The framer specifies the width to use for fixed border overlap in the Method Preference window. An example would be a design in which four boards are used and all three the bottom boards are cut at a width of 40mm.


3. Custom Overlap

When a board is added to a design, it is possible to define custom overlaps for the particular board. This is done by selecting the custom overlap checkbox when the Add Board dialog is displayed. Enter the preferred overlaps and click on OK.


4. Full size

All the outside measurements off all the boards are the same as that of the top board.


5. Full bottoms

For all the above methods, it can be specified to use full bottoms. In this case the bottom border of each board is extended to the outermost board's bottom edge.
