How to add new Suppliers to RSF

Terminology

Material items are the materials that make up a design. If a design consists of two mount boards, one frame, one back and glass, the design will contain five material items. Each material item has specific attributes like supplier, colour, length and width, which defines the material item.

Adding suppliers to the RSF supplier database has no influence on the stock control system.

Adding Suppliers

  • Navigate to Materials | Suppliers on the main toolbar.

  • The suppliers window will be displayed.

  • Click on the button to add a new supplier. All the data entry fields will be cleared.
