RFS backup and restore functions

1. Backing up the database

To backup the database, select File | Backup. At the prompt, select yes to start the procedure.

2. Restore the database

To restore a backup database, select File | Restore. A list of earlier backup files will be displayed in the format “BU” + date of backup. Choose the file you want to restore and click the restore button.

NOTE: Make sure that ALL designs are closed when running the restore database procedure.
