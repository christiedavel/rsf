Capturing of information

Before any designs can be created, the information which will be used in the design needs to be captured. The most important of these are the material and supplier information without which it is impossible to create a design. All the data capturing windows follow the same procedure to capture and edit the information and are controlled by two toolbars, the data tool bar and the scrolling tool bar.

The data capturing windows are:

  1. Client and Artist

    These windows can be found under the Artists and Clients menu. For clients, the minimum information that must be supplied are the first and last names. For artists, only a last name is required. All the other information fields are optional.

  2. Suppliers

    To view the supplier window, navigate to Materials | Suppliers. A unique supplier name must be supplied. All other fields are optional.

  3. Materials

    Materials will be covered in the materials section.
